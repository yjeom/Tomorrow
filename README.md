# Tomorrow
 spring web project
 본 스프링 웹 프로그램은 가장 큰 기능은 2가지로 이루어져있다. 
첫번째는,익명 1:1 고민상담이다. 이 기능은 편지처럼 1:1로 나눌 수 있으며 고민을 작성하고 전송하면 다른 회원 중 한 사람에게 랜덤으로 전송됨으로 사용자들로부터 아날로그적인 펜팔 감성을 느끼게 하고 진정성 있는 고민을 나누게 해줄 수 있다.  
두번째는, 365가지의 질문에 대한 대답과 그날 하루를 마무리하며 가장 좋았던 일과 그렇지 않았던 일 그리고 내일의 소원을 기록할 수 있는 기능이다. 365가지의 질문과 그날의 세줄 일기의 기능은 사용자로 하여금 자신에 대해서 깊게 알게 하고, 힐링을 느끼게 한다.
